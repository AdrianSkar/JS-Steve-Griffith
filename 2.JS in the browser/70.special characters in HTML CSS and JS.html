Special characters in HTML, CSS & JS

Keyboards have a limited number of characters. We still need to be able to display characters that are not on our keyboards. Sometimes we are putting the characters in HTML, sometimes we use CSS to load content. Other time we use JavaScript to write out new content in the HTML. All three have a different way of adding the characters.

HTML character entities (decimal): &#165; → ¥

HTML character entities (hexadecimal): #x00A5; → ¥

HTML character entities (abbreviation): &yen; → ¥
