Getting started with Babel

Original install methods result in npm vulnerabilities. Use npm i --save-dev @babel/cli @babel/preset-env instead.

Additionally you need to delete .babelrc and use babel.config.json instead. More at Using Babel.

If you delete the node_modules folder or want to run it from a different device, you can copy the package.json file and run npm install.
